    - `IRGraphClient.get_graph(graph_id)` returns graph JSON on 200, None on 404.
    - `IRGraphClient.create_graph(name)` sends POST to `/api/graphs`, ensuring "(agent)" is appended to the name if missing.
    - `IRGraphClient.add_node(graph_id, node_data)` sends POST to `/api/graphs/{graph_id}/nodes`.
    - `IRGraphClient.add_edge(graph_id, edge_data)` sends POST to `/api/graphs/{graph_id}/edges`.
    - `IRGraphClient.update_node(graph_id, node_id, node_data)` sends PUT to `/api/graphs/{graph_id}/nodes/{node_id}`.
    - `IRGraphClient.delete_node(graph_id, node_id)` sends DELETE to `/api/graphs/{graph_id}/nodes/{node_id}`.
    - All methods raise `RuntimeError` with "IR Graph API unreachable" message if `httpx.ConnectError` occurs.
    - `IRGraphClient.is_agent_owned(graph_id)` returns True if graph name contains "(agent)".
